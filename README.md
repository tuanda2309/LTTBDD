## üõ†Ô∏è Tech Stack

This collection demonstrates the use of standard Android development tools and libraries:

* **Language:** **Kotlin** (Primary language).
* **UI Design:**
    * **XML Layouts:** Classic Android view system.
    * **Material Design:** Utilizing Material Components for UI.
* **Architecture:**
    * **MVVM Pattern:** (Applied in later weeks/Elearning project).
    * **ViewBinding:** For safer interaction with views.
* **Jetpack Libraries:**
    * **Lifecycle & ViewModel:** Managing UI-related data.
    * **Navigation Component:** Handling in-app navigation.
* **Backend & Data:**
    * **Firebase:** Authentication and Cloud Firestore.
    * **REST API:** Consuming external data services.
    * **Glide/Picasso:** Image loading and caching.

## üöÄ Getting Started

1.  Clone this repository to your local machine.
2.  Open **Android Studio**.
3.  Select **"Open an existing Android Studio project"**.
4.  Navigate to the specific week folder (e.g., `LTTBDD_Tuan3`) and open it.
5.  Sync Gradle and Run the app on an Emulator or Physical Device.
